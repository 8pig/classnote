## JDBC驱动
    * 定义了操作所有关系型数据库的规则(接口)
    * jdbc 本质: 官方定义的一套操作所有关系型数据库的规则, 即接口. 各个数据库厂商区实现这个套接口提供数据库驱动jar包
-----
## JDBC 入门
    * 步骤
        1. 导入驱动jar 包
            1. 复制 jar 包到目录下
            2. add as library
        2. 注册驱动
        3. 获取数据库连接对象 connection  不能直接执行
        4. 定义sol
        5. 获取执行sql语句的对象 statement
        6. 执行sql 接受返回结果
        7. 处理结果
        8. 释放资源
