# 查询
* select * from 表名;

    1. 语法
        select
            字段列表
        from
            表名列表
        group by
            分组字段
        having
            分组之后的条件
        order by
            排序
        limit
            分页
    2. 基础查询
        - 查询多个字段
            * select 列名1, 列名2, 列名n from 表名;
            * 注意
                * 如果查询所有字段 用* 代替
        - 去重
            * distinct
        - 计算列
            * 一般可以使用四则运算计算列值, 一般只进行数值计算
            * ifnull(表达式1, 表达式2); null运算都为null
                * 表达式1: 表示哪个字段需要判断是否为null
                * 表达式2: 字段为null 替换的值
        - 起别名
            * as
